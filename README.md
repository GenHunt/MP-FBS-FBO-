# Ozon FBS Label Printer

Локальное Windows-приложение для автоматизированной печати этикеток по заказам Ozon FBS со встроенным редактором шаблонов.

## 🎯 Основные возможности

- ✅ **Интеграция с Ozon API** - получение списка заказов в сборке
- ✅ **Печать маршрутных этикеток** - вывод оригинальных этикеток Ozon
- ✅ **Печать внутренних этикеток** - гибкий редактор шаблонов для создания пользовательских этикеток
- ✅ **Система переменных** - подстановка артикула, названия, штрихкода и других данных из API
- ✅ **Управление шаблонами** - создание, редактирование, копирование и удаление шаблонов
- ✅ **Пакетная печать** - вывод этикеток для нескольких заказов одной кнопкой
- ✅ **Журнал логирования** - отслеживание всех операций печати
- ✅ **Поддержка Xprinter XP-365B** - оптимизирована для работы с термопринтером

## 📋 Требования

- **Python 3.10+**
- **Windows 7+**
- **Принтер Xprinter XP-365B** (или другой совместимый)
- **Доступ к Ozon Seller API**

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/GenHunt/MP-FBS-FBO-.git
cd MP-FBS-FBO-
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Конфигурация

Скопируйте `.env.example` в `.env` и заполните свои данные:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
OZON_CLIENT_ID=your_client_id_here
OZON_API_KEY=your_api_key_here
PRINTER_NAME=Xprinter XP-365B
```

Где получить ключи API:
- Перейдите на https://seller.ozon.ru/
- Откройте личный кабинет → Настройки → Доступ к API
- Скопируйте Client ID и API Key

### 4. Запуск приложения

```bash
python main.py
```

## 📖 Как пользоваться

### Первый запуск

1. Запустите приложение
2. В верхней части окна введите **Client ID** и **API Key**
3. Нажмите кнопку **"✓ Подключиться"**
4. После успешного подключения статус должен измениться на **"✅ Подключено к Ozon"**

### Получение заказов

1. Перейдите на вкладку **"📦 Заказы"**
2. Нажмите кнопку **"🔄 Обновить список заказов"**
3. В таблице появятся все заказы Ozon FBS в сборке
4. Выберите заказы, которые нужно печатать (галочки в первой колонке)

### Редактирование шаблонов

1. Перейдите на вкладку **"✏️ Редактор шаблонов"**
2. Выберите шаблон слева или создайте новый кнопкой **"➕ Новый"**
3. Добавьте элементы:
   - **📝 Текстовые элементы** - для названия, артикула и т.д.
   - **📊 Штрихкод** - для печати кода товара
4. Настройте свойства каждого элемента (размер, позиция, шрифт)
5. Нажмите **"💾 Сохранить шаблон"**

### Печать этикеток

1. Перейдите на вкладку **"🖨️ Печать"**
2. Выберите тип печати:
   - **Только маршрутные этикетки** - печать оригинальных этикеток Ozon
   - **Только внутренние этикетки** - печать по выбранному шаблону
   - **Маршрутные + внутренние** - полный набор этикеток
3. Выберите шаблон для внутренних этикеток
4. Нажмите:
   - **"🖨️ Печать выбранных"** - для печати только отмеченных заказов
   - **"🖨️ Печать всех"** - для печати всех заказов в списке

### Просмотр логов

1. Перейдите на вкладку **"📋 Логи"**
2. Выберите файл лога из списка
3. Просмотрите детали операций печати

### Настройки приложения

1. Перейдите на вкладку **"⚙️ Настройки"**
2. Измените параметры принтера и приложения
3. Нажмите **"💾 Сохранить"**

## 📊 Доступные переменные для шаблонов

При редактировании шаблона этикетки доступны следующие переменные для подстановки:

| Переменная | Описание | Пример |
|-----------|---------|--------|
| `article` | Артикул товара | `ABC123456` |
| `product_name` | Название товара | `Товар XYZ` |
| `barcode` | Штрихкод (для печати) | `4606362021234` |
| `sku` | SKU товара | `123456789` |
| `offer_id` | Offer ID на Ozon | `987654321` |
| `quantity` | Количество товара | `2` |
| `shipment_id` | Номер отправления | `1234567-001` |
| `order_id` | Номер заказа | `987654321` |
| `manufacturer_part_number` | Партномер производителя | `MPN-12345` |
| `price` | Цена товара | `1299.99` |
| `created_at` | Дата создания | `2026-06-15T10:30:00` |

## 🔧 Разработка

### Структура проекта

Проект использует следующие основные технологии:

- **PyQt6** - графический интерфейс
- **Requests** - HTTP клиент для Ozon API
- **Pillow** - обработка изображений
- **python-barcode** - генерация штрихкодов
- **ReportLab** - создание PDF

### Добавление новых переменных в шаблон

1. Отредактируйте модель данных в `src/models/shipment.py`
2. Добавьте переменную в метод `to_dict()` класса `ShipmentItem` или `Shipment`
3. Обновите список переменных в `src/ui/tabs/template_editor_tab.py` в методе `update_variables_list()`

### Добавление нового типа элемента шаблона

1. Добавьте тип в `ElementType` в `src/models/template.py`
2. Реализуйте метод рисования в `LabelGenerator` в `src/printing/__init__.py`
3. Добавьте поддержку в редакторе шаблонов `template_editor_tab.py`

## 🐛 Решение проблем

### Ошибка подключения к API

- Проверьте правильность Client ID и API Key
- Убедитесь, что у вас есть доступ к Ozon Seller API
- Проверьте интернет-соединение

### Принтер не печатает

- Убедитесь, что принтер Xprinter XP-365B включен и подключен
- Проверьте название принтера в настройках Windows
- Обновите имя принтера в файле `.env` или в вкладке "Настройки"

### Этикетки печатаются с неправильным форматом

- Отредактируйте шаблон в редакторе шаблонов
- Проверьте размеры элементов (в миллиметрах)
- Убедитесь, что выбран правильный размер этикетки (по умолчанию 58x40 мм)

## 📝 Логирование

Приложение логирует все операции в файл `logs/ozon_label_printer.log`:

- Печать этикеток
- Ошибки и исключения

Просмотреть логи можно во вкладке **"📋 Логи"** приложения.
