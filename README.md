# Ozon FBS Label Printer

Локальное Windows-приложение для автоматизированной печати этикеток по заказам Ozon FBS со встроенным редактором шаблонов и **аналитикой отслеживания времени поступления заказов**.

## 🎯 Основные возможности

- ✅ **Интеграция с Ozon API** - получение списка заказов в сборке
- ✅ **Печать маршрутных этикеток** - вывод оригинальных этикеток Ozon
- ✅ **Печать внутренних этикеток** - гибкий редактор шаблонов для создания пользовательских этикеток
- ✅ **Система переменных** - подстановка артикула, названия, штрихкода и других данных из API
- ✅ **Управление шаблонами** - создание, редактирование, копирование и удаление шаблонов
- ✅ **Пакетная печать** - вывод этикеток для нескольких заказов одной кнопкой
- ✅ **📊 Аналитика отгрузок** - отслеживание времени поступления заказов и оптимизация отгрузок
- ✅ **Журнал логирования** - отслеживание всех операций печати
- ✅ **Поддержка Xprinter XP-365B** - оптимизирована для работы с термопринтером

## 📊 Новая функция: Аналитика отгрузок

### Что это дает?

Встроенная система аналитики помогает **оптимизировать время отгрузки** на основе исторических данных:

- 📈 **Распределение заказов по часам** - узнайте, когда приходит больше всего заказов
- 📅 **Паттерны по дням недели** - определите самые загруженные дни
- 💡 **Оптимальные окна отгрузки** - система автоматически рекомендует лучшее время
- 🔮 **Прогнозы на неделю** - предсказание загруженности
- ⚠️ **Оценка риска задержек** - риск-ассессмент текущей нагрузки
- 📊 **Коэффициент эффективности** - общая оценка производительности

### Как использовать?

1. Перейдите на вкладку **"📊 Аналитика"**
2. Выберите период анализа (7, 30, 60 или 90 дней)
3. Нажмите **"🔄 Обновить данные"**
4. Изучите 5 вкладок с разными взглядами на данные:
   - **Эффективность** - общие метрики производительности
   - **По часам** - распределение заказов в течение дня
   - **По дням** - паттерны по дням недели
   - **Рекомендации** - оптимальные окна отгрузки
   - **Прогнозы** - предсказание на следующую неделю

## 📋 Требования

- **Python 3.10+**
- **Windows 7+**
- **Принтер Xprinter XP-365B** (или другой совместимый)
- **Доступ к Ozon Seller API**

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/GenHunt/MP-FBS-FBO-.git
cd MP-FBS-FBO-
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Конфигурация

Скопируйте `.env.example` в `.env` и заполните свои данные:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
OZON_CLIENT_ID=your_client_id_here
OZON_API_KEY=your_api_key_here
PRINTER_NAME=Xprinter XP-365B
```

### 4. Запуск приложения

```bash
python main.py
```

## 📖 Как пользоваться

### Первый запуск

1. Запустите приложение
2. В верхней части окна введите **Client ID** и **API Key**
3. Нажмите кнопку **"✓ Подключиться"**
4. После успешного подключения статус должен измениться на **"✅ Подключено к Ozon"**

### Получение заказов

1. Перейдите на вкладку **"📦 Заказы"**
2. Нажмите кнопку **"🔄 Обновить список заказов"**
3. В таблице появятся все заказы Ozon FBS в сборке
4. Выберите заказы, которые нужно печатать (галочки в первой колонке)

### Анализ времени отгрузок

1. Перейдите на вкладку **"📊 Аналитика"**
2. Выберите период анализа (рекомендуется минимум 30 дней данных)
3. Нажмите **"🔄 Обновить данные"**
4. Изучите каждый раздел:
   - **Эффективность** - насколько хорошо вы обрабатываете заказы
   - **По часам** - видите, когда приходит больше всего работы
   - **По дням** - определите самые сложные дни недели
   - **Рекомендации** - получите точные временные окна для отгрузки
   - **Прогнозы** - подготовьтесь к предсказываемым пикам

### Печать этикеток

1. Перейдите на вкладку **"🖨️ Печать"**
2. Выберите тип печати
3. Выберите шаблон
4. Нажмите **"🖨️ Печать выбранных"**

## 📊 Работа с аналитикой

### Ключевые метрики

| Метрика | Описание |
|---------|---------|
| **Коэффициент эффективности** | 0-100 баллов для оценки вашей работы |
| **Доля вовремя обработанных** | Процент заказов обработанных за 24 часа |
| **Время обработки** | Среднее время от получения до отгрузки |
| **Пиковые часы** | Часы с максимальной очередью |
| **Загруженность по дням** | Какие дни самые сложные |

## 📝 Доступные переменные для шаблонов

| Переменная | Описание |
|-----------|---------|
| `article` | Артикул товара |
| `product_name` | Название товара |
| `barcode` | Штрихкод |
| `sku` | SKU товара |
| `offer_id` | Offer ID |
| `quantity` | Количество |
| `shipment_id` | Номер отправления |
| `order_id` | Номер заказа |
| `manufacturer_part_number` | Партномер производителя |
| `price` | Цена товара |
| `created_at` | Дата создания |

## 🐛 Решение проблем

### Ошибка подключения к API

- Проверьте правильность Client ID и API Key
- Убедитесь, что у вас есть доступ к Ozon Seller API
- Проверьте интернет-соединение

### Принтер не печатает

- Убедитесь, что принтер Xprinter XP-365B включен и подключен
- Проверьте название принтера в настройках Windows
- Обновите имя принтера в файле `.env` или в вкладке "Настройки"

### Аналитика не загружается

- Убедитесь, что у вас есть минимум несколько дней исторических данных
- Проверьте логи в вкладке "Логи"
