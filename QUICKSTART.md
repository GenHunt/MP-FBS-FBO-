# 🚀 Быстрый старт

Хотите быстро начать работу? Вот минимальные шаги!

## 1️⃣ Установка (5 минут)

```bash
# Клонируйте репозиторий
git clone https://github.com/GenHunt/MP-FBS-FBO-.git
cd MP-FBS-FBO-

# Установите зависимости
pip install -r requirements.txt

# Скопируйте конфиг
copy .env.example .env
```

## 2️⃣ Конфигурация (2 минуты)

Откройте файл `.env` и заполните:

```env
OZON_CLIENT_ID=ваш_client_id
OZON_API_KEY=ваш_api_key
PRINTER_NAME=Xprinter XP-365B
```

**Где получить ключи?**
- Перейдите на https://seller.ozon.ru/
- Откройте личный кабинет → Настройки → Доступ к API
- Скопируйте Client ID и API Key

## 3️⃣ Запуск (1 минута)

```bash
python main.py
```

## 4️⃣ Первый тест (3 минуты)

1. Введите Client ID и API Key в верхней части окна
2. Нажмите **"✓ Подключиться"**
3. Перейдите на вкладку **"📦 Заказы"**
4. Нажмите **"🔄 Обновить список заказов"**
5. Выберите заказ и нажмите **"🖨️ Печать выбранных"**

## 📚 Что дальше?

- **Редактор шаблонов**: Вкладка **"✏️ Редактор шаблонов"**
- **Настройки**: Вкладка **"⚙️ Настройки"**
- **Логи**: Вкладка **"📋 Логи"**
- **Полная документация**: Посмотрите `README.md`
- **Детальная установка**: Посмотрите `INSTALL.md`

## 🆘 Быстрое решение проблем

| Проблема | Решение |
|----------|---------|
| "python is not recognized" | Переустановите Python с опцией "Add to PATH" |
| Ошибка подключения к API | Проверьте Client ID и API Key |
| Принтер не печатает | Проверьте название принтера в Windows |
| "No module named PyQt6" | Выполните `pip install -r requirements.txt` |

## 🎯 Популярные задачи

### Изменить размер этикетки

1. Вкладка **"✏️ Редактор шаблонов"**
2. Выберите шаблон
3. Измените значения `width` и `height` в элементах
4. Нажмите **"💾 Сохранить шаблон"**

### Добавить новое поле в этикетку

1. Вкладка **"✏️ Редактор шаблонов"**
2. Нажмите **"📝 Добавить текст"**
3. Выберите переменную в раскрывающемся списке
4. Расположите элемент
5. Нажмите **"💾 Сохранить шаблон"**

### Печать только для одного заказа

1. На вкладке **"📦 Заказы"** снимите все галочки
2. Отметьте нужный заказ
3. Нажмите **"🖨️ Печать выбранных"**

### Просмотр истории печати

1. Вкладка **"📋 Логи"**
2. Выберите файл лога
3. Просмотрите операции
